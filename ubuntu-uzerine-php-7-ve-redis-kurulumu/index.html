Ubuntu Üzerine Php 7 ve Redis Kurulumu

PHP 7 yayınlanalı çok oluyor yeni sürüme geçmeyenler yada hali hazırda php5-fpm kullananlar olabilir nginx üzerinde sıkıntı çekeceğini düşünenler olabilir.Kurmayan varsa kurmalarını öneriyorum niye derseniz php5 sürümüne göre oldukça hızlı tabi hala HHVM kadar hızlı değil ancak php den vazgeçemeyen arkadaşlar için oldukça ideal

Grafiksel olarak göstermek gerekirse Php 7 Wordpress de olukça performanslı çalışıyor.

Tabi burada saf php7 gösterilmiş durumda biz bu kuruluma birde redis eklediğimizde otomatikmen HHVM performansına ulaşmış olacağız tabi HHVM ve redis kurulumu yapıldığında dahada performanslı bir sistemimiz olacak
